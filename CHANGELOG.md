# 変更履歴

このファイルには、プロジェクトの重要な変更内容が記録されます。

## [1.0.0] - 2025-12-17

### 🎉 初回リリース

#### ✨ 追加機能

**生徒・保護者向けアプリ**
- ログイン機能（メール+パスワード認証）
- ホーム画面（お知らせ・イベント表示）
- 掲示板機能（コルクボード風デザイン）
- カレンダー機能（イベント一覧）
- メッセージ受信機能
- レスポンシブデザイン対応

**管理者向けツール**
- ダッシュボード（統計情報表示）
- 投稿管理（作成・削除）
- メッセージ管理（送信・閲覧・削除）
- ユーザー管理（追加・削除・権限設定）
- イベント管理（登録・削除）
- PC最適化UI

**バックエンド**
- Firebase Authentication連携
- Firestore Database統合
- リアルタイム更新機能
- 権限ベースのセキュリティルール

**デプロイ**
- Vercel対応設定
- セキュリティヘッダー設定
- 静的ファイル最適化

#### 📚 ドキュメント

- README.md（プロジェクト概要）
- SETUP_GUIDE.md（詳細セットアップガイド）
- USER_GUIDE.md（使い方ガイド）
- firestore.rules（セキュリティルール）
- vercel.json（デプロイ設定）

#### 🔧 技術仕様

- **フロントエンド**: HTML5, Tailwind CSS, JavaScript ES6+
- **バックエンド**: Firebase (Auth, Firestore, Storage)
- **ホスティング**: Vercel
- **アイコン**: Font Awesome 6
- **フォント**: Noto Sans JP

#### 🎯 対象規模

- 生徒: 約30名
- 職員: 約10名
- 完全無料枠で運用可能

## バージョン番号について

このプロジェクトは[セマンティックバージョニング](https://semver.org/lang/ja/)に従います：

- **MAJOR**: 互換性のない変更
- **MINOR**: 後方互換性のある機能追加
- **PATCH**: 後方互換性のあるバグ修正
